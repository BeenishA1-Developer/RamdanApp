# Ramadan Tasbeeh App

### Overview
This is a modern Android application built using **Jetpack Compose** and **Room Database**. It provides a digital Tasbeeh counter for Ramadan with automated daily resets and persistent storage.

### Features
- **Digital Tasbeeh Counter:** Track your dhikr with a sleek, interactive UI.
- **Daily Auto-Reset:** Built using **WorkManager** to reset counts at midnight.
- **Local Persistence:** Data remains safe across app restarts via **Room DB**.
- **AI Assisted:** Developed using advanced AI-assisted architecture and design optimization.

### Tech Stack
- Kotlin
- Jetpack Compose
- Room DB
- WorkManager
- Material 3 Design
